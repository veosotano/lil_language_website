@container

This class represents an object in the dom tree that has position and size.

Example code:

@root { #new @container test { width: 100; height: 60; background: #0; } }

Member variables

  • id

    Type: var.i64

    A number that represents this unique container instance

  • boxId

    Type: vvar.i64

    The id of the box that is associated with this container

  • width

    Type: vvar.f32

  • height

    Type: vvar.f32

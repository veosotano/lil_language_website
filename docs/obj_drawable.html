@drawable

This is an abstract class that is shared by any object that can be drawn on the screen.

It is implemented as a class snippet that needs to be pasted.

Example code:

class @myClass { #paste @drawable; //etc }; var.@myClass myObj: @{ name: "myDrawable"; }; myObj.x: 100.0; myObj.y: 30.5;

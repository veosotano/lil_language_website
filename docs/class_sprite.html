@sprite

A sprite is an image which clips its content to a given box.

This class extends the @image class with the clipping functionality

Example code

#new @sprite character {
	width: 64;
	height: 64
	src: "sprite.png";
	x: 20;
	y: 10;
	clipX: 0;
	clipY: 128;
}

Virtual variables

  • clipX

    Type: vvar.f32

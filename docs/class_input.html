@input

This class has storage for the states of the keys of the keyboard, the mouse and gamepads.

Example code

var gamepad: input.getGamepad();

if input.getMousePressed() {
	someObj.setX(input.getMouseX());
	someObj.setY(input.getMouseY());
}

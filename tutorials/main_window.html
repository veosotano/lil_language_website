The main window

The topmost element is called root, and its width and height define the size of the window on those platforms where it is supported, like Mac, Windows or Linux. On other platforms like iOS or Android the size of the window is defined by the operating system and these values will be ignored. The background color fills the entire screen with the given color.

@root { width: 800.0; height: 600.0; background: #00C53A; }
